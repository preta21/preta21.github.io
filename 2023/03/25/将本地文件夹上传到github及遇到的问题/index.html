Ubuntu下先安装 git

  1. git密钥的新申请方法

    从 2020 年之后，github不再支持远端的密码访问
    必须通过密钥访问。且不支持 RSA 密钥，而是使用 ed25519 密钥

如果没有公钥文件，则如下申请

  1. 将本地的文件夹上传远端 git

    注意一点，如今 github 默认的主分支名称是 main 而不是 master
    而在本地建的仓库默认主分支名称是 master，需要解决该冲突，见第 3 点

  1. 如果遇到本地的主分支为 master 而远程主分支为 main 的情况（反之亦然），需要在本地新建 main分支，删除本地以及远程 master 分支，并将本地 main 分支推送到远端 main。
